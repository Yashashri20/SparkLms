
Learning Management System (LMS)

1.Project Overview

The **Learning Management System (LMS)** is a web-based platform designed to simplify the teaching and learning process. It allows teachers to manage educational content and students to access learning materials anytime, anywhere.

This project focuses on providing a **simple, user-friendly, and efficient system** compared to complex platforms.

---

2.bjectives

* Provide a clean and responsive user interface
* Allow teachers to upload lectures, notes, and announcements
* Enable students to access study materials easily
* Conduct quizzes with automatic scoring
* Maintain student attendance records
* Store all data securely using a database

---

3.Tech Stack

**Frontend:**

* HTML
* CSS 
* JavaScript

**Backend:**

* Node.js

**Database:**

* MySQL

---

4.Features

### Teacher Module

* Secure login
* Upload lectures (videos/documents)
* Upload notes (PDF/Images)
* Create quizzes (auto-evaluated)
* Post announcements
* Mark student attendance

### Student Module

* Registration & login
* View lectures and notes
* Attempt quizzes
* View scores
* Check announcements

---
## Database Design

The system uses multiple tables such as:

* Users (Teacher/Student)
* Courses
* Quizzes
* Quiz Scores
* Notes
* Announcements
* Attendance

---

5.Scope

* Payment gateway integration
* Offline access
* Multi-language support
* Advanced gamification

---
